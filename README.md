# TradesbyPinsel

TradesbyPinsel is a simple Minecraft trading plugin that allows players to trade items with each other.
Players can list items for a price, and other players can buy them directly.

## Features

* Player-to-player trading
* List items for a custom price
* Easy to use
* Lightweight and simple

## How it works

Players can put items up for sale and set a price.
Other players can browse available trades and purchase the items.

This makes trading on the server easier and safer.

## Installation

1. Download the plugin `.jar`
2. Put it into your server **plugins** folder
3. Restart the server

## Compatibility

* Minecraft Server
* Works with Paper / Spigot based servers
