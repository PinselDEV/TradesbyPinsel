TradesbyPinsel

A simple and intuitive Minecraft trading plugin that allows players to sell and buy items from each other using a GUI system.

TradesbyPinsel provides a lightweight marketplace where players can list items for a custom price and other players can purchase them easily.

Features

Simple GUI-based trading system

Players can sell items for a custom price

Players can browse items listed by other players

Manage and remove your own listings

Admin GUI for managing and removing listings

Fast and lightweight system

Easy to use for both players and administrators

How the Plugin Works

When a player runs the command /trade, a GUI opens with three options:

Sell Item

My Listings

Buy Items

Each option opens a new GUI where players can interact with the trading system.

Selling Items

When selecting Sell Item, a trading GUI opens.

Players can place the item they want to sell into the center slot.

The price can be adjusted using buttons:

+1

+10

+100

+1000

-1

-10

-100

-1000

Right-clicking the buttons increases or decreases the value faster.

In the middle of the GUI a paper item shows the current price and information about the trade.

After setting the desired price, players can confirm the listing using the green confirmation button.

Players can cancel the process using the cancel button.

Buying Items

When selecting Buy Items, players can browse all currently listed items.

Players can:

view items listed by other players

purchase items directly from the GUI

The system automatically handles the transaction.

My Listings

The My Listings menu allows players to manage their active listings.

Players can:

view all items they currently have listed

remove their listings using right-click

Admin Menu

Administrators have access to an additional admin function inside the GUI.

Admins will see a special red block that allows them to manage listings from other players.

Admins can remove listings if necessary.

Player Commands

/trade
Opens the trading GUI.

Admin Commands

/trade reload
Reloads the plugin configuration.

Requirements

Minecraft Server

Spigot or Paper

Installation

Download the plugin .jar

Place it into your server plugins folder

Restart the server

Performance

TradesbyPinsel is designed to be lightweight and efficient.

The plugin does not heavily impact server performance and works smoothly even on larger servers.
